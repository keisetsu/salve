Class: HashSet

hashstructs~ HashSet

A set of objects. The objects are distinguished by a hash function.

new HashSet(hash_f, initial)

Parameters:
Name Type Argument Description
hash_f function

A function which returns a uniquely identifying hash when called with an object that a HashSet instance uses. Note that it is a valid implementation strategy for the hash function to know how to handle only a certain type of object, and not everything under the sun. This entails that a HashSet object using this hash function can only contain objects of the type that the hash function knows how to handle.
initial Object <optional>

An initial value for the object being constructed.

<private> _store(hash, val)

Record a hash and value pair into the backing store. Effectively associates the hash with the value. This method assumes but does not verify that the mapping from hash to value is unique. This method cannot be used to change such mapping.
