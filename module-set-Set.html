Class: Set

set~ Set

This is a naive implementation of sets. It stores all elements in an array. All array manipulations are done by searching through the array from start to hit. So when adding a new element to the Set for instance, the add method will scan the whole array, find the element is not there and then add the element at the end of the array. As naive as this implementation is, it has been shown to be faster than HashSet and when used in the context of this library.

Note that Set cannot hold undefined values.
